# CharteCY

## Description
Ce projet est une application web présentant la charte graphique de CY. Elle permet de visualiser les directives de marque de manière interactive.

## Technologies utilisées
- React
- Vite
- Framer Motion
- Lucide React

## Installation et Lancement
1. Installer les dépendances :
   ```bash
   npm install
   ```
2. Lancer l'application en mode développement :
   ```bash
   npm run dev
   ```
3. Construire pour la production :
   ```bash
   npm run build
   ```

## Déploiement
Le projet est déployé automatiquement via GitHub Pages.
